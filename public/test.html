📝 All Available Configuration Options:

Display & Layout:
  • data-mode: chat | voice | both
  • data-theme: light | dark | auto
  • data-position: bottom-right | bottom-left | top-right | top-left
  • data-size: small | medium | large | full
  • data-border-radius: none | small | medium | large
Colors:
  • data-base-bg-color: #ffffff
  • data-accent-color: #9333ea
  • data-cta-button-color: #000000
  • data-cta-button-text-color: #ffffff
Text Content:
  • data-title: Button text
  • data-start-button-text: Start button
  • data-end-button-text: End button
  • data-chat-first-message: Welcome msg
  • data-chat-placeholder: Input placeholder
  • data-agent-display-name: Agent name
Features:
  • data-custom-image-url: Logo URL
  • data-help-texts: Comma-separated list
  • data-voice-show-transcript: true/false
  • data-consent-required: true/false
  • data-language-selector-enabled: true/false
  • data-languages: JSON array

✨ Why embed.js (Iframe Approach)?

• All icons work perfectly (uses React + Lucide icons)
• Complete isolation - No CSS/JS conflicts with your site
• Fully transparent when collapsed - No visible borders
• Non-blocking - Doesn't interfere with page clicks
• Easy updates - Update widget without touching client sites
• Proven approach - Used by Intercom, Drift, LiveChat
• Single file - Just include embed.js
